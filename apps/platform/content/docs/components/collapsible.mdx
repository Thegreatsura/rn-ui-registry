## Usage

<DocSection>
  Use `Collapsible` when the user should stay on the same surface while revealing optional context.

  <DocSubsection>
    <CompPreview
      code={`<Collapsible defaultOpen>
  <CollapsibleTrigger asChild>
    <Button variant="outline">Toggle details</Button>
  </CollapsibleTrigger>
  <CollapsibleContent>
    <Text variant="muted">Optional supporting content.</Text>
  </CollapsibleContent>
</Collapsible>`}
    >
      <CollapsibleInlinePreview />
    </CompPreview>
  </DocSubsection>
</DocSection>
