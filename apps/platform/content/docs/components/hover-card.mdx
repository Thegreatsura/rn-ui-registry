## Usage

<DocSection>
  On native, tap the trigger to open and tap again to close. On the platform
  preview, pointer hover uses Radix Hover Card.

### Basic

<DocSubsection>
  <CompPreview
    code={`<HoverCard>

## API Reference

<DocSection>
  `HoverCardContent` supports `align` and `sideOffset` on web (Radix). Native
  content is absolutely positioned under the trigger, similar to `Popover`.
</DocSection>
