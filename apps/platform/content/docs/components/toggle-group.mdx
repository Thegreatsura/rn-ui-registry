## Usage

<DocSection>
  Use toggle groups when a set of view or platform options should share the same visual affordance.

  <DocSubsection>
    <CompPreview
      code={`<ToggleGroup type="multiple" defaultValue={["ios"]}>
  <ToggleGroupItem value="ios">iOS</ToggleGroupItem>
  <ToggleGroupItem value="android">Android</ToggleGroupItem>
  <ToggleGroupItem value="web">Web</ToggleGroupItem>
</ToggleGroup>`}
