## Usage

<DocSection>
  Compose items inside `PaginationContent`. Use `PaginationLink` with
  `active` for the current page.

### Basic

<DocSubsection>
  <CompPreview
    code={`<Pagination>
  <PaginationContent>
    <PaginationItem>
      <PaginationPrevious onPress={() => {}} />
    </PaginationItem>
    <PaginationItem>
      <PaginationLink active>1</PaginationLink>
    </PaginationItem>
  </PaginationContent>
</Pagination>`}
  >
    <PaginationInlinePreview />
  </CompPreview>
</DocSubsection>
</DocSection>

## API Reference

<DocSection>
  `PaginationLink` accepts `active` and `disabled`. On web, links use
  `onClick`; on native, `onPress` from `Pressable`.
</DocSection>
