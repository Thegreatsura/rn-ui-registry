## Usage

<DocSection>
  Radio groups let the user choose exactly one option from a related set.

  <DocSubsection>
    <CompPreview
      code={`<RadioGroup value={value} onValueChange={setValue}>
  <RadioGroupItem value="starter">
    <Text>Starter</Text>
  </RadioGroupItem>
  <RadioGroupItem value="pro">
    <Text>Pro</Text>
  </RadioGroupItem>
</RadioGroup>`}
