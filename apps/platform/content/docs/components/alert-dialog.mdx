## Usage

<DocSection>
  Use an alert dialog for high-consequence decisions.

  <DocSubsection>
    <CompPreview
      code={`<AlertDialog>
  <AlertDialogTrigger asChild>
    <Button variant="outline">Reset analytics</Button>
  </AlertDialogTrigger>
  <AlertDialogContent>
    <AlertDialogHeader>
      <AlertDialogTitle>Reset analytics?</AlertDialogTitle>
      <AlertDialogDescription>
        Existing charts will clear and rebuild from the next event.
      </AlertDialogDescription>
    </AlertDialogHeader>
    <AlertDialogFooter>
      <AlertDialogCancel asChild>
        <Button variant="ghost">Keep data</Button>
      </AlertDialogCancel>
      <AlertDialogAction asChild>
        <Button variant="destructive">Reset</Button>
      </AlertDialogAction>
    </AlertDialogFooter>
  </AlertDialogContent>
</AlertDialog>`}
    >
      <AlertDialogInlinePreview />
    </CompPreview>
  </DocSubsection>
