## Usage

<DocSection>
  Provide an icon (or glyph) as `children` and set `accessibilityLabel`. Sizes include `sm`, `default`, and `lg`. Use `variant="secondary"` for a softer surface.
  <DocSubsection>
    <ApiTable rows={[{ prop: "size", type: "\"sm\" | \"default\" | \"lg\"", default: "\"default\"", description: "Diameter of the FAB." }, { prop: "variant", type: "\"default\" | \"secondary\"", default: "\"default\"", description: "Primary vs secondary fill." }]} />
  </DocSubsection>
  For a speed dial, use **`FabMenu`**: pass `actions` (`id`, `label`, optional `onPress` and `icon`) and optionally **`renderMain`** so the main button switches between open and closed icons. The menu uses an in-area dimmed backdrop; give the preview container enough height (about `220px`) so actions stack above the main FAB.
  <DocSubsection>
    <ApiTable rows={[{ prop: "actions", type: "FabMenuAction[]", default: "—", description: "Stacked secondary actions with labels." }, { prop: "renderMain", type: "(open: boolean) => ReactNode", default: "+ / ×", description: "Main FAB content for open vs closed." }, { prop: "accessibilityLabel", type: "string", default: "\"Open actions menu\"", description: "Main FAB when closed." }]} />
  </DocSubsection>
</DocSection>
