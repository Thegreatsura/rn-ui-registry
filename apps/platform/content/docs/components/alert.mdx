## Usage

<DocSection>
  Compose `AlertTitle` and `AlertDescription` inside `Alert`. Use `variant="destructive"` for errors.

### Basic

<DocSubsection>
  <CompPreview
    code={`<Alert>

</Alert>`}

## API Reference

<DocSection>
  `Alert` accepts a `variant` prop (`default` | `destructive`). Title and
  description inherit semantic colors from the variant on native and web.
</DocSection>
